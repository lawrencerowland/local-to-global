# Local to Global

A public workbench for small, self-contained demonstrations that move from local evidence to global structure.

## Foray 001

**Sheaf models for PMO scope coherence** is an interactive static page showing how local work-package scope claims can be checked across interfaces before being glued into a coherent programme view.

The site is intentionally static: `index.html` is the whole experience, so GitHub Pages can serve it directly from the repository root.

## Foray 002

**Sheaf project director explainer** is a second static prototype in the series, kept under `forays/sheaf-project-director-explainer.html`.
